What is grid based design?

It is dividing the webpage into even sections of columns to form a grid, which is then useful to help structure elements on the page. Sections of a grid and have smaller grids within them also.

What's all the hype about responsive webpages?

People access the internet on so many different kinds of devices, which have different sized screens. Webpages need to be able to change and adapt to any sized screen, so the user still has a good experience of the page.

What is semantic structure?

Organising and structuring html code so that tags reflect the differing importance of the elements on the page.
