Javascript Language.

Parentheses ()

Parentheses are used as part of a conditional statement, normally containing the conditional as part of an if/else statement, or in a loop.

They are also used in functions, holding the parameters that the function takes in - function(parameter).

You can also use them to group expressions together so that they are computed as a whole before being included in the rest of the expression, such as in mathematics. E.g. (5 + 6) * 8

Brackets []

Brackets are used to hold arrays - E.g. myArray = [1,2,3]

They are also used to then access particular values in an array using indices - E.g. myArray[1]

Braces {}

Braces are generally used to contain certain sections of code into blocks, such as the code in a for loop or in an if statement. They also contain the code executed by a function.

Braces are also used to form objects, containing all the keys and values of the object.

Single '' and Double "" Quotes

Single and double quotes are generally interchangeable. They are used to denote that the information contained within them is of type string.

If you want quotation marks to appear in the string, then you need to use one type of quotes as the outer pair, and then the other type as the ones in the string.
